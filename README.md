# TOC Project 2020 - 新聞小幫手

## 簡介

此小幫手整合了四家新聞網站 每個網站有四種不同的分類:社會、國際、政治、運動 讓使用者能隨時隨地知道最新的新聞

## 流程

1. 輸入想看的媒體
2. 輸入想看的種類
3. 顯示結果
